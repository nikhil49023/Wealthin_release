## üöÄ Quick Start Guide

### Import Transactions from SMS
1. Go to **Transactions** screen
2. Tap **Import from SMS**
3. Grant SMS permission if not already done
4. Wait for automatic scanning (may take 1-2 minutes for 5000+ messages)
5. Review and categorize imported transactions

### Create a DPR (Detailed Project Report)
1. Go to **Brainstorm Canvas**
2. Add your business ideas using canvas items
3. Select AI Mode: **Strategic Planner** for market analysis
4. Switch to **Financial Architect** for projections
5. Generate DPR section-by-section
6. Export as PDF when complete

### Use AI Advisor
1. Go to **AI Advisor** screen
2. Select mode based on your need:
   - **Strategic Planner**: Business strategy questions
   - **Financial Architect**: Financial modeling
   - **Execution Coach**: Implementation planning
3. Ask questions or request analysis
4. Save important insights to canvas

## üîß Troubleshooting

### App Won't Install
- Ensure Android version is 5.0 or higher
- Enable "Unknown Sources" in Settings ‚Üí Security
- Check available storage (need 200MB+)

### SMS Not Importing
- Grant SMS permission in app settings
- Ensure bank SMS are in inbox
- Try manual refresh

### Contact Names Not Showing
- Grant Contacts permission
- Ensure contacts are saved with phone numbers
- Restart app to reload contact cache

### PDF Export Not Working
- Grant Storage permission
- Check available storage space
- Try exporting to different location
