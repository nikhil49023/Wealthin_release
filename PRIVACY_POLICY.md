# Wealthin - Privacy Policy

**Last Updated:** March 25, 2026

## 1. Data Collection & Usage

Wealthin **collects ONLY essential financial data** to provide its core services:

### Data We Collect:
- **Financial Transactions**: Amounts, dates, categories (locally stored)
- **User Profile**: Name, email, account details (for authentication)
- **Documents**: PDFs of bank statements, receipts (uploaded for analysis)
- **Notification Content**: Bank transaction notifications (processed locally)

### Data We DO NOT Collect:
- ❌ SMS messages or call logs (except bank notifications)
- ❌ Contact lists or call history
- ❌ Browsing history or app usage patterns
- ❌ Location data
- ❌ Device identifiers or MAC addresses

## 2. Data Storage

- **Local Storage**: All sensitive financial data encrypted locally on your device using Android Keystore
- **Cloud Sync** (Optional): Only synced to Firebase if user explicitly enables backup
- **API Keys**: Stored securely in encrypted storage, never logged or transmitted unnecessarily

## 3. API Integrations

Wealthin uses **only one external AI provider**:

### Sarvam AI
- **Purpose**: Document analysis, financial advice, transaction categorization
- **Data Sent**: Only the minimum necessary (transaction descriptions, document content)
- **Privacy**: Sarvam has explicit NO data retention agreements for this integration
- **Link**: https://www.sarvam.ai/privacy

## 4. Data Retention

- **Transaction Data**: Kept until user deletes
- **Uploaded Documents**: Kept until user clears
- **API Logs**: Kept for 30 days by Sarvam (then deleted)
- **User Account**: Deleted on account termination + 90 day grace period

## 5. User Rights

You have the right to:
- ✅ **Access** all your data - export via Settings > Export Data
- ✅ **Delete** any transaction or document anytime
- ✅ **Download** all your data in CSV/JSON format
- ✅ **Opt-out** of cloud sync at any time
- ✅ **Unlink API keys** and use offline mode

## 6. Security

- All data encrypted in transit (HTTPS) and at rest
- No third-party analytics or tracking
- Regular security audits
- Automatic logout after 15 minutes of inactivity

## 7. Changes to Policy

We will notify users of material changes via in-app notification and email 30 days in advance.
