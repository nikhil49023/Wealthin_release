## 🔧 Quick Code Reference

### Import
```dart
import 'package:wealthin_flutter/core/theme/luxury_colors.dart';
import 'package:wealthin_flutter/core/widgets/glassmorphic.dart';
```

### Basic Usage
```dart
// Color
color: LuxuryColors.peachCream

// Gradient
gradient: LuxuryColors.sunriseLuxury

// Glass Container
GlassContainer(
  gradient: LuxuryColors.mintBreeze,
  child: YourWidget(),
)

// Glass Card
Container(
  decoration: LuxuryColors.luxuryGlassCard(),
  child: YourWidget(),
)
```

## 💡 Pro Tips

1. **Warm + Cool Balance**: Mix warm (Peach, Golden) with cool (Mint, Olive)
2. **Depth Layers**: Use 3 shades for depth (light, medium, dark)
3. **Accent Sparingly**: Golden Sand at 30-40% for borders
4. **Text Contrast**: Always check with `getTextColor()`
5. **Gradient Direction**: Top-left to bottom-right for depth
6. **Glass Opacity**: Lower for backgrounds (0.1-0.2), higher for cards (0.7-0.9)
