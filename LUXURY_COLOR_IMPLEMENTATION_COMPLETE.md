## 🚀 How to Use

### Quick Start (5 minutes)
```dart
// 1. Import
import 'package:wealthin_flutter/core/theme/luxury_colors.dart';
import 'package:wealthin_flutter/core/widgets/glassmorphic.dart';

// 2. Use colors
color: LuxuryColors.peachCream

// 3. Apply glass effect
GlassContainer(
  gradient: LuxuryColors.mintBreeze,
  child: YourWidget(),
)

// 4. Use gradients
gradient: LuxuryColors.sunriseLuxury
```

## 🔧 Troubleshooting

### Common Issues & Solutions

**Issue**: Colors not showing
- **Solution**: Check import statements, rebuild app

**Issue**: Glass effect not visible
- **Solution**: Verify blur values (10-20), check opacity (0.1-0.3)

**Issue**: Text not readable
- **Solution**: Use `LuxuryColors.getTextColor()` or `AppTheme.textPrimary(context)`

**Issue**: Performance lag
- **Solution**: Reduce blur radius, limit glass effects on low-end devices
